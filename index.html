Sesión 3

Analítica Avanzada con QUERY
Concepto Clave
La función QUERY permite ejecutar análisis complejos con sintaxis tipo SQL directamente en Google Sheets, transformando datos crudos en información procesable mediante consultas estructuradas.

Cláusulas Principales de QUERY

SELECT
Selecciona qué columnas retornar y en qué orden. Si se omite, todas las columnas de la tabla son retornadas en su orden predeterminado
WHERE
Retorna únicamente las filas que coinciden con una condición. Si se omite, todas las filas son retornadas
GROUP BY
Agrega valores a través de filas, permitiendo consolidar datos por categorías específicas
PIVOT
Transforma valores distintos en columnas en nuevas columnas, creando tablas dinámicas para análisis multidimensional
ORDER BY
Ordena filas por valores en columnas, permitiendo clasificación ascendente o descendente
LIMIT
Limita el número de filas retornadas, útil para muestras o resúmenes de datos grandes
OFFSET
Omite un número dado de primeras filas, permitiendo paginación de resultados
LABEL
Establece etiquetas de columnas personalizadas para mejorar la presentación de resultados
FORMAT
Formatea los valores en ciertas columnas usando patrones de formato específicos
OPTIONS
Establece opciones adicionales para personalizar el comportamiento de la consulta

Sintaxis y Estructura

Orden de las Cláusulas
La sintaxis del lenguaje de consulta está compuesta de las siguientes cláusulas. Cada cláusula comienza con una o dos palabras clave. Todas las cláusulas son opcionales. Las cláusulas se separan por espacios. El orden de las cláusulas debe ser el siguiente:

SELECT → WHERE → GROUP BY → PIVOT → ORDER BY → LIMIT → OFFSET → LABEL → FORMAT → OPTIONS
